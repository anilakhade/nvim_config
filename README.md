# nvim_config

Setup details: Arch Linux, i3wm, Neovim.  
Main languages: Python and C++.

How to use:

1. Install Neovim: `sudo pacman -S neovim`
2. Place the config file at: `~/.config/nvim/init.lua`
3. Open Neovim (`nvim`). All plugins will install automatically.
